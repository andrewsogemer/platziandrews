Guía de Uso - Sistema de Control Migratorio

Información: Esta es una guía paso a paso para usar el Sistema de Control Migratorio. Abre index.html para acceder al sistema.

Paso 1: Acceder al Sistema

Para comenzar a usar el sistema:

  1. Abre el archivo index.html en tu navegador
  2. El sistema se cargará automáticamente con datos de ejemplo
  3. Verás el dashboard principal con estadísticas
  4. Navega entre las pestañas usando los botones superiores

Paso 2: Registrar una Persona

Para registrar una nueva persona:

  1. Ve a la pestaña "Registro"
  2. Completa todos los campos marcados con (*)
  3. Ingresa datos como nombre, documento, nacionalidad, etc.
  4. Selecciona el motivo del viaje
  5. Haz clic en "Registrar Persona"

Campos requeridos:

  • Nombre Completo
  • Número de Documento
  • Tipo de Documento
  • Nacionalidad
  • Fecha de Nacimiento
  • Género
  • Motivo del Viaje

Paso 3: Registrar Entrada/Salida

Para registrar movimientos migratorios:

  1. En la pestaña "Registro"
  2. Ingresa el número de documento de la persona
  3. Si es una persona nueva, completa todos los datos
  4. Haz clic en "Registrar Entrada" o "Registrar Salida"
  5. El sistema actualizará automáticamente el estado

Estados posibles:

  • Entrada: Persona que acaba de ingresar
  • Salida: Persona que acaba de salir
  • Pendiente: Persona registrada pero sin movimiento

Paso 4: Buscar Personas

Para encontrar personas registradas:

  1. Ve a la pestaña "Búsqueda"
  2. Escribe en el campo de búsqueda
  3. Puedes buscar por:
    • Nombre de la persona
    • Número de documento
    • Nacionalidad
  4. Los resultados se muestran automáticamente
  5. Haz clic en "Ver" para ver detalles completos

Paso 5: Generar Reportes

Para crear reportes y estadísticas:

  1. Ve a la pestaña "Reportes"
  2. Selecciona el tipo de reporte:
    • Entradas por Fecha: Todas las entradas en una fecha
    • Salidas por Fecha: Todas las salidas en una fecha
    • Por Nacionalidad: Estadísticas por país
    • Por Motivo de Viaje: Análisis de motivos
    • Por Estado: Distribución de estados
  3. Selecciona una fecha si es necesario
  4. Haz clic en "Generar Reporte"

Paso 6: Configurar el Sistema

Para personalizar el sistema:

  1. Ve a la pestaña "Configuración"
  2. Modifica los parámetros según tu necesidad:
    • País de Origen: País donde opera el sistema
    • Punto de Control: Nombre del punto migratorio
    • Usuario: Usuario actual del sistema
    • Fecha del Sistema: Configuración de fecha/hora
  3. Haz clic en "Guardar Configuración"
  4. Los cambios se guardan automáticamente

Dashboard

Vista general con estadísticas en tiempo real, incluyendo total de personas, entradas y salidas del día, y personas pendientes.

Validación

Sistema de validación automática que verifica campos requeridos, previene duplicados y asegura la integridad de los datos.

Historial

Registro completo de todos los movimientos migratorios con fechas, observaciones y detalles de cada entrada y salida.

Responsivo

Interfaz adaptativa que funciona perfectamente en computadoras, tablets y dispositivos móviles.

Consejos de Uso

  • Búsqueda rápida: Usa el campo de búsqueda en el dashboard para encontrar personas rápidamente
  • Validación: Asegúrate de completar todos los campos requeridos marcados con (*)
  • Reportes: Genera reportes regularmente para mantener un control de las estadísticas
  • Backup: Los datos se guardan automáticamente en tu navegador
  • Navegación: Usa las pestañas para acceder a las diferentes funcionalidades
